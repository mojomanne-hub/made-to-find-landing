# Made to Find Landingpage

Eine responsive Next.js-Landingpage für Made to Find mit Tailwind CSS und Framer Motion.

## Starten

```bash
npm install
npm run dev
```

Danach ist die Seite unter `http://localhost:3000` erreichbar.

## Original-Screenshots einsetzen

Die Komponente `components/app-screen.tsx` enthält bewusst stilisierte App-Mockups, damit die Landingpage sofort ohne Bildmaterial funktioniert. Sobald die Original-Screenshots verfügbar sind, können diese gezielt dort oder als Bilder in den Hero- und Schritt-Karten eingesetzt werden.

## Wichtige Dateien

- `app/page.tsx` – Seitenaufbau und alle Inhaltsbereiche
- `app/globals.css` – Design, Responsivität und Animationen
- `components/app-screen.tsx` – wiederverwendbare App-Ansichten
